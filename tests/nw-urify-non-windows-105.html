nw-urify-non-windows-105

Resolve a string with an implicit scheme with a relative path against a relative file URI.

Test nw-urify-non-windows-105.xml is expected to fail with error code err:XD0074.

It requires the following features: urify-non-windows.

The pipeline


<p:declare-step xmlns:err="http://www.w3.org/ns/xproc-error" xmlns:p="http://www.w3.org/ns/xproc" name="pipeline" version="3.0">
   <p:output port="result">
   <p:identity>
      <p:with-input>
         <result>{p:urify("Documents and Files/thing",
                             "file:not-absolute")}</result>
      </p:with-input>
   </p:identity>
</p:declare-step>
