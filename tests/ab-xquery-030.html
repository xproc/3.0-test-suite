p:xquery 030 (AB)

Tests p:xquery: JSON arrays can not be source docs in XQuery 1.0

Test ab-xquery-030.xml is expected to fail with error code err:XC0101.

It requires the following features: xquery_1_0.

The pipeline


      <p:declare-step xmlns:p="http://www.w3.org/ns/xproc" version="3.0">
         <p:output port="result"/>
         
         <p:xquery version="1.0">
            <p:with-input port="source">
               <p:inline content-type="application/json">["one", "two"]</p:inline>
            </p:with-input>
            <p:with-input port="query">
               <p:inline content-type="text/plain">
                  declare namespace array="http://www.w3.org/2005/xpath-functions/array";
                  <result>{{array:get(., 2)}}</result>
               </p:inline>
            </p:with-input>
         </p:xquery>
      </p:declare-step>
