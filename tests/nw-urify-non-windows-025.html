nw-urify-non-windows-025

Resolve a string with an implicit scheme with an absolute path against an absolute http: URI.

Test nw-urify-non-windows-025.xml is expected to pass.

It requires the following features: urify-non-windows.

The pipeline


<p:declare-step xmlns:p="http://www.w3.org/ns/xproc" name="pipeline" version="3.0">
   <p:output port="result">
   <p:identity>
      <p:with-input>
         <result>{p:urify("///path/to/thing",
                             "http://example.com/documents/")}</result>
      </p:with-input>
   </p:identity>
</p:declare-step>

Schematron validation


<s:schema xmlns="http://www.w3.org/1999/xhtml" xmlns:s="http://purl.oclc.org/dsdl/schematron" queryBinding="xslt2">
   <s:pattern>
      <s:rule context="/">
         <s:assert test="result">Root element is not 'result'.</s:assert>
         <s:assert test="string(result)= 'http://example.com/path/to/thing'">Incorrect URI in result.</s:assert>
      </s:rule>
   </s:pattern>
</s:schema>
