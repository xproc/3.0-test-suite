nw-urify-non-windows-060

Resolve a string with an explicit file URI scheme with a relative path against a non-hierarchical urn URI.

Test nw-urify-non-windows-060.xml is expected to fail with error code err:XD0080.

It requires the following features: urify-non-windows.

The pipeline


<p:declare-step xmlns:err="http://www.w3.org/ns/xproc-error" xmlns:p="http://www.w3.org/ns/xproc" name="pipeline" version="3.0">
   <p:output port="result">
   <p:identity>
      <p:with-input>
         <result>{p:urify("file://authority.com",
                             "urn:publicid:ISO+8879%3A1986:ENTITIES+Added+Latin+1:EN")}</result>
      </p:with-input>
   </p:identity>
</p:declare-step>
