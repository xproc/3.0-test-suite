nw-urify-windows-161

Resolve a string with an explicit file URI scheme with a relative path on drive C: against a relative file URI.

Test nw-urify-windows-161.xml is expected to fail with error code err:XD0074.

It requires the following features: urify-windows.

The pipeline


<p:declare-step xmlns:err="http://www.w3.org/ns/xproc-error" xmlns:p="http://www.w3.org/ns/xproc" name="pipeline" version="3.0">
   <p:output port="result">
   <p:identity>
      <p:with-input>
         <result>{p:urify("file:C:Users/Jane/Documents and Files/Thing",
                             "file:not-absolute")}</result>
      </p:with-input>
   </p:identity>
</p:declare-step>
